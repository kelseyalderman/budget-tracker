# Budget Tracker

## Description

A Progressive Web Application with offline access and functionality that allows users to log expenses and deposits to their budget with or without an internet connection.

## Built With

- HTML
- CSS
- Bootstrap
- JavaScript
- Node.js
- Express.js
- IndexedDB
- Mongoose
- MongoDB
- Compression
- Morgan
- Heroku
